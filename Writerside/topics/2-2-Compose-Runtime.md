# 2.2 Compose Runtime

Compose Runtime은 Jetpack Compose의 핵심 구성 요소이며, 프로그래밍 모델과 상태 관리의 핵심 엔진 역할을 합니다.
Jetpack Compose는 기본 구성 요소에 대한 깊은 지식 없이도 직관적이고 원활하게 작동하도록 설계되었지만,
Compose Runtime API를 이해하는 것은 성능 및 메모리 관리를 최적화하는 데 중요합니다.

Compose Runtime은 기본적으로 내부에서 상태를 관리하여 수동 개입의 필요성을 줄입니다. 
그러나 사용하는 API가 내부적으로 어떻게 작동하는지 확실히 이해하면, 의도하지 않은 부작용을 최소화하면서 
특히 상태 관리 및 부작용 처리를 다룰 때 더 효율적인 애플리케이션을 구축하는 데 도움이 될 수 있습니다.
